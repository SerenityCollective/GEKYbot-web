GEKYbot

GEKYbot is a general purpose Discord Bot written in the Discord.js library. GEKYbot is a fairly
simple Discord Bot but new features are immninent weekly.

Invite GEKYbot

Commands

  • ;doggo | Display random.dog image
  • ;ban @user reason | Ban a guild member
  • ;kick @user reason | Kick a guild member
  • ;warn @user reason | Warn a guild member
  • ;report @user reason (optional image) | Report a member of the guild
  • ;clear num | Bot deletes num of messages
  • ;avatar @user | Display a user's avatar
  • ;mute @user reason | Mute a user
  • ;question xyz? | Bot answers question
  • ;say xyz | Bot says specified text
  • ;serverinfo | Display server info
  • ;botinfo | Display info about GEKYbot
  • ;ping | Check bot latency and API latency
  • ;tempmute 1s/1m/1w/1d | Temp mute a user
  • ;dev | Display GEKYbot developer
  • ;eval | Execute JS code
  • ;kitty | Show random.cat image
  • ;warnlevel | Show user warnings
  • ;help | Bot DMs help menu
  • ;iag | Agree to rules in official bot server only.
  • ;coins | Check your coins
  • ;level | Check your level
  • ;addrole @user rolename | Give a user a role.
  • ;remove role @user rolename | Remove a role.
  • ;unmute @user | Unmute an user.
  • ;pay @user amount | Give a user coins

Some of these commands are not functioning and each require their own permission from the message author.

All issues can be reported here, or on our Discord.
